This python application scrapes data from the following url:
https://parivahan.gov.in/rcdlstatus/?pur_cd=101

The python libraries used are:
1. requests
2. lxml
3. pandas
4. json

Make sure you install these libraries before running the Python program!

When the app asks for captcha, see the .png file in the folder and enter the captcha respectively.
